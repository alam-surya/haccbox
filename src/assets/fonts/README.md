# Satoshi Font Installation

## Cara Install Font Satoshi

1. **Download Font**
   - Kunjungi: https://font.download/font/satoshi
   - Klik tombol "Download Font for Free"
   - Ekstrak file ZIP yang didownload

2. **Place Font Files**
   - Copy semua file font (`.woff2`, `.woff`, atau `.ttf`) ke folder ini: `src/assets/fonts/`
   - Pastikan nama file sesuai dengan yang ada di `fonts.css`:
     - `Satoshi-Light.woff2` (atau `.woff` / `.ttf`)
     - `Satoshi-LightItalic.woff2`
     - `Satoshi-Regular.woff2`
     - `Satoshi-Italic.woff2`
     - `Satoshi-Medium.woff2`
     - `Satoshi-MediumItalic.woff2`
     - `Satoshi-Bold.woff2`
     - `Satoshi-BoldItalic.woff2`
     - `Satoshi-Black.woff2`
     - `Satoshi-BlackItalic.woff2`

3. **Format File**
   - Prioritas: `.woff2` > `.woff` > `.ttf`
   - Jika file yang didownload menggunakan nama berbeda, rename sesuai dengan nama di atas

4. **Usage**
   - Font sudah dikonfigurasi sebagai default font family di Tailwind CSS
   - Gunakan class Tailwind seperti biasa: `font-sans`, `font-light`, `font-medium`, `font-bold`, dll
   - Atau gunakan CSS: `font-family: 'Satoshi', sans-serif;`

## Catatan Lisensi

Font Satoshi adalah **gratis untuk penggunaan personal**. Untuk penggunaan komersial, hubungi Indian Type Foundry.
